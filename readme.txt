== Description ==

= RUS =
Администратору потребуется найти копирайтера фрилансера, после чего для нового «члена команды» будет создана учетная запись, посредством вноса данных email, произвольных логина/пароля. Далее копирайтер сможет без проблем заходить под созданной для него учетной записью в панель администрирования. Автор будет фактически осуществлять наполнение сайтов контентом, готовить материал, публиковать данные. 
Более подробну информация вы можете почитать на [домашней странице](http://avkproject.ru/plugins/accounting-records-copywriter.html) плагина.

= ENG =
Admin would have to hire freelance copywriter and after a login account shall be made for the new “team member” by entering e-mail data and undefined login/password. Thus copywriter will have smooth access to administration bar under created login account. Author will in fact fill sites with content, prepare materials, and post data.
Detailed information can be found on [plugin page](http://avkproject.ru/plugins/accounting-records-copywriter.html#tabs-lang-2).

== Installation ==

1. Upload `accounting-records-copywriter` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. That's all.
